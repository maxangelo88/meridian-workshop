Architectural notes & gaps

  • No database — all data loaded from JSON at server startup; writes are not persisted across restarts gap
  • Task management endpoints (GET/POST/PATCH/DELETE /api/tasks) are stubs — frontend calls fail silently gap
  • Purchase order creation endpoint is a stub — backlog workflow is incomplete gap
  • No authentication or authorisation on any endpoint — open CORS + no API keys gap
  • No global state library (Pinia/Vuex) — shared state managed via composables intentional
  • Custom SVG charts — no external charting dependency (Chart.js, etc.) intentional
  • i18n implemented without a library — locale + currency switching via useI18n.js composable delivered
  • Restocking recommendation logic is computed server-side on each request — no caching layer gap
